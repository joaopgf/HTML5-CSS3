Principais formatações

Negrito / Destaque

Nesta frase há um termo em negrito usando a tag B (não semântica)

Nesta frase há um termo em destaque usando a tag strong (semântica)

Itálico / Ênfase

Nesta frase há um termo em itálico usando a tag i (não semântica)

Nesta frase há um termo em ênfase usando a tag EM (semântica)

Texto marcado

podemos também marcar um texto usando a tag MARK

Texto grande e pequeno

Podemos deixar textos grandes e pequenos.

Texto deletado

Podemos mostrar um texto como deletado usando a tag del

Texto inserido

podemos marcar um texto como inserido usando a tag ins (semântica)

E também usando a tag U (não semântica)

Texto sobrescrito

Para marcar um texto sobrescrito usamos a tag sup. EX.: X3+5

Texto subscrito

Para marcar um texto sobrescrito usamos a tag sub. EX.: H2O

Blockquote

No livro PHP: a Bíblia, Tim Converse explica o que é PHP.

O PHP é uma linguagem para a criação de scripts para a Web do lado servidor embutidos em HTML, cujo código-font é aberto, e que é compatível com os mais importantes servidores Web (especialmente Apache). O PHP permite incorporar fragmentos de código em páginas HTML normais - código esse que é interpretado à medida que suas páginas são oferecidas aos usuários. O PHP também serve como uma linguagem que "cola", facilitando a conexão de suas páginas Web com o banco de dados do lado servidor.
